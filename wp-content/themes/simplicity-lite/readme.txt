Simplicity, Simple but Powerful WP Theme

== Description ==

Simplicity is a CSS3 and HTML5 Powered WordPress Latest Version Ready Theme for Small Business Companies. Simplicity is Responsive, Super Elegant and Professional Theme which will be adjusted automatically with all smart devices. You can Visit the Simplicity Demo Here: http://demo.d5creation.com/wp/themes/simplicity

== Frequently Asked Questions ==

= What to do after Theme Installation ? =
After installing the Theme Please go to Appearance > Customize. You can set the Slide Images, Featured Boxes and Featured Contents etc. there

= Why is Simplicity? =
Simplicity is Super Easy, User Friendly, Elegant, Responsive and Professional

= Can it be used for Personal or Company Websites? =
Yes, Of Course

= Is it only for Blogs ? =
No, It is for Blogs, Company, Personal too all kind of Websites
